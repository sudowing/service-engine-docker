# service-engine-docker
Dockerized Service-Engine Service

## QUICK START

### Set Secrets

Set vars in `.env` (env.template provided)
```
cp env.template .env
```
### Run App
```
npm run start
```

http://localhost:8081/openapi
http://localhost:8081/service-engine-app/graphql/

### OpenAPI UI
```
npm run api-docs
```

### Run Waddershins Markdown
```
npm run make:api-md
```
